Abstract base class for general VQE algorithms¶

vqeabc.py¶

The abstract base class inherited by subclasses that execute general variational quantum eigensolvers.

class qforte.abc.vqeabc.VQE(system, reference, trial_state_type='reference', trotter_order=1, trotter_number=1, fast=True, verbose=False, print_summary_file=False)[source]¶
_ompimizer¶

The type of optimizer to use for the classical portion of VQE. Suggested algorithms are ‘BFGS’ or ‘Nelder-Mead’ although there are many options (see SciPy.optimize.minimize documentation).

Type

string

_converged¶

_final_result¶

Type

object

_opt_maxiter¶

The maximum number of iterations for the classical optimizer.

Type

int

_opt_thresh¶

The numerical convergence threshold for the specified classical optimization algorithm. Is usually the norm of the gradient, but is algorithm dependant, see scipy.minimize.optimize for detials.

Type

float
